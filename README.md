# Data Insights using Java

## Project Overview
This project analyzes a customer churn dataset using Java and generates meaningful insights such as churn rate, gender-based churn, and spending patterns.

## Features
- Total customer count
- Churn rate calculation
- Gender-wise churn analysis
- High spending customer detection

## Technologies Used
- Java
- File Handling
- Data Processing

## How to Run
1. Place CSV file in project root
2. Run MainApp.java
