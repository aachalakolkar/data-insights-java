# Customer Churn Analysis using Java

## 📌 Project Overview
This project analyzes a real-world customer churn dataset using Java. It processes raw CSV data and generates meaningful business insights such as churn rate, demographic trends, and spending behavior.

## ⚙️ Features
- Total customer count
- Churn rate calculation
- Gender-wise churn analysis
- High spending customer identification

## 📊 Sample Output
Total Customers: 1000  
Churn Rate: 23.5%  
Male Churn: 120  
Female Churn: 115  
High Spending Customers: 300  

Insight: Male customers are churning slightly more than female customers.

## 🗂️ Dataset
The dataset contains customer details such as age, gender, visit behavior, ratings, and churn status. It is used to analyze customer retention patterns.

## 🛠️ Technologies Used
- Java
- File Handling
- Data Processing

## ▶️ How to Run
1. Place the CSV file in the project root directory  
2. Run `MainApp.java`
